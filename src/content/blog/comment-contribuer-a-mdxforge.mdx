# Comment contribuer à MDXForge

MDXForge est un projet open-source passionnant qui vise à simplifier la création de sites web avec Astro et MDX. Voici comment vous pouvez contribuer :

1. **Fork le repository** : Commencez par créer votre propre copie du projet sur GitHub.

2. **Clonez votre fork** : Téléchargez le code sur votre machine locale.

3. **Créez une nouvelle branche** : Travaillez sur vos modifications dans une branche séparée.

4. **Faites vos modifications** : Améliorez le code, la documentation ou ajoutez de nouvelles fonctionnalités.

5. **Testez vos changements** : Assurez-vous que tout fonctionne correctement.

6. **Soumettez une Pull Request** : Proposez vos modifications à l'équipe principale.

Nous apprécions toutes les contributions, grandes ou petites !
